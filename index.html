⚠️ Requisito: CORS en Apps Script
Para que la sincronización funcione entre dispositivos, tu Apps Script debe incluir los headers CORS en doGet y doPost. Asegúrate de que tu Codigo.gs tenga estas líneas en doGet: 
function doGet(e){
  var output = ContentService
    .createTextOutput(JSON.stringify(handleGet(e)))
    .setMimeType(ContentService.MimeType.JSON);
  return output;
}
function doPost(e){
  var output = ContentService
    .createTextOutput(JSON.stringify(handlePost(e)))
    .setMimeType(ContentService.MimeType.JSON);
  return output;
}
Despliega siempre como "Ejecutar como: yo" y "Quién tiene acceso: Cualquier persona". Cada vez que modifiques el script debes crear una nueva versión de implementación (no "Editar la misma").
